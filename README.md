
# Exercícios de Python – Módulo 3  
**Programação de Sistemas Informáticos**

Este repositório contém os exercícios desenvolvidos em **Python**, no âmbito do **Módulo 3** da disciplina **Programação de Sistemas Informáticos**, do **Curso Profissional Técnico de Gestão e Programação de Sistemas Informáticos**.

Os exercícios abordam conceitos fundamentais da programação, desde instruções básicas até estruturas de controlo, ciclos, listas e funções.

---

## 📚 Conteúdos Abordados

### ✅ Introdução ao Python
- Utilização da função `print()`
- Declaração e uso de variáveis
- Entrada de dados com `input()`
- Conversão de tipos (`int`, `float`, `str`)

---

### ✅ Operadores e Expressões
- Operações aritméticas (soma, subtração, multiplicação e divisão)
- Operador módulo (`%`)
- Comparações lógicas
- Verificação de condições simples

---

### ✅ Estruturas de Decisão
- Instrução `if`
- Estruturas `if / elif / else`
- Avaliação de condições múltiplas
- Exemplos práticos:
  - Verificação de maioridade
  - Classificação de notas
  - Comparação de números
  - Autenticação de utilizador
  - Cálculo de descontos
  - Verificação de múltiplos
  - Ano bissexto

---

### ✅ Estruturas de Repetição
- Ciclo `while`
- Ciclo `for`
- Utilização de contadores
- Exemplos práticos:
  - Contagens crescentes e decrescentes
  - Números pares
  - Tabuada
  - Soma de conjuntos de números
  - Cálculo de fatorial

---

### ✅ Listas
- Criação e manipulação de listas
- Adição de elementos
- Percurso de listas
- Cálculo de média
- Listagem de valores com base em condições

---

### ✅ Programas com Menus
- Implementação de menus interativos
- Estrutura de escolha com opções
- Repetição até saída do utilizador

---

### ✅ Funções
- Definição de funções
- Passagem de parâmetros
- Organização de código
- Projeto prático:
  - Sistema de Gestão de Stocks
    - Registo de materiais
    - Consulta de stock
    - Atualização de quantidades
    - Exibição do estado geral do stock

---

## 🛠 Tecnologias Utilizadas
- Python 3
- Ambiente de linha de comandos (CLI)

---

## 🎯 Objetivo Pedagógico
O objetivo destes exercícios é consolidar:
- Lógica de programação
- Estruturas fundamentais do Python
- Desenvolvimento de programas interativos
- Boas práticas de organização de código
